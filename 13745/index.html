¿Qué aprendimos en la unidad?

Durante esta unidad, aprendimos los fundamentos de la computación en la nube, incluyendo el uso de plataformas como Microsoft Azure para desplegar y administrar recursos tecnológicos de forma remota.

Uno de los enfoques principales fue la creación y gestión de máquinas virtuales (VMs). Aprendimos a:

  • Configurar una cuenta en Azure.
  • Crear máquinas virtuales desde el portal de Azure.
  • Seleccionar el sistema operativo, tamaño y región adecuados para nuestras necesidades.
  • Conectar a las VMs mediante RDP (Windows) o SSH (Linux).
  • Gestionar el ciclo de vida de las máquinas: inicio, detención, eliminación y monitoreo.
